# Simple HSV Sliders

A simple HSV sliders color selection plugin for Krita, adapted for PyQt5 / PyQt6 dual compatibility.

## 兼容性说明

本插件兼容 **Krita 5.0 及以上版本**（支持 PyQt5 与 PyQt6 双引擎环境）。
